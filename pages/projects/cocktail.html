Cocktail Index App

A React and Bootstrap 5 based cocktail index SPA(Single Page Application), with search feature.
This app is powered by TheCocktailDB API and deployed on Netlify.

    Tech Stack:
  • React.js,
  • React Router
  • React useContext API
  • Bootstrap 5
  • Font Awesome
  • GIT
  • Bash
  • NPM

Key Features
